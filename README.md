# Strong & Radiant 💛

A personal 16-week fitness & nutrition PWA — knee-safe, home equipment, built for real life.

## Features
- 📅 3-Phase workout plan (15–30 min/day, knee-friendly)
- 🔥 Nutrition targets & daily habit guide
- 🍽 7-day meal plan with macros
- 🛒 Interactive grocery checklist
- ✅ Workout log with streak tracking
- 📈 Weight & water intake tracker
- 💛 Knee care guide

## Dev Setup

```bash
npm install
npm run dev
```

## Deploy to Netlify (one-time setup)

1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → New site from Git
3. Connect your GitHub repo
4. Set build settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
5. Click Deploy

Every push to `main` will auto-deploy. ✓

## Install as App on Phone

- **iPhone:** Open site in Safari → Share → Add to Home Screen
- **Android:** Open in Chrome → ⋮ menu → Add to Home Screen
